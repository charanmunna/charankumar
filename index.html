Tittle :-

3-Phase Reactive Power Compensation using STATCOM(Static Synchronous Compensator) device through MATLAB Software.

Description :-

This project presents a study on MATLAB/Simulink model of STATCOM(static synchronous compensator) for reactive power compensation. The STATCOM is shunt connected facts device used for improve voltage profile of transmission system which connected line with voltage source inverter with dc link capacitor. Reactive power compensation using STATCOM is used for reducing power loss and improves power factor and voltage profile in transmission system. It is the best way to overcome the problem of reactive power compenation without constructing new transmission lines.
